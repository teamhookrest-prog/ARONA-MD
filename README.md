Base Script WhatsApp Tipe ESM yang ringan, cepat, stabil, dan modern. Dibangun dengan Baileys berbasis ECMAScript Modules (ESM), mudah dipahami, serta fleksibel untuk dikembangkan lebih lanjut.


---

🚀 Fitur Utama

Fitur	Status	Keterangan

Simple Code	✅	Mudah dibaca & dikembangkan
Pairing With Code	✅	Login tanpa scan QR
Multi-Device Support	✅	Stabil digunakan di HP/PC
Plugin System	✅	Tambah fitur hanya dengan drop file
ESM Modules	✅	Menggunakan standar JavaScript modern
Auto Reconnect	✅	Tetap online 24/7
Database JSON	✅	Menyimpan data user & grup
LID System	✅	Sistem alternatif pengganti JID
Button Support	❌	Keterbatasan Baileys terbaru



---

📋 Instalasi Script

1. Software yang diperlukan :

Git

Node.js v20+

ImageMagick

FFmpeg


2. Deploy Script :

git clone https://github.com/teamhookrest-prog/ARONA-MD.git

cd ARONA-MD
npm i
npm start
